Session c

IRI http://parliament.uk/ontologies/written-statement/Session
Is Defined By pws:
Description

A period during which Parliament is able to conduct business on days of its choosing. A session is within a Parliament period. The first session of a Parliament begins on the same day as the Parliament period and ends on a day named in a subsequent prorogation proclamation. Subsequent sessions within the Parliament period begin and end on dates named in subsequent prorogation proclamations. The final session of a Parliament ends either with prorogation followed by dissolution or with dissolution. Session periods are the same for both Houses.
In Range Of pws:inSession op

Expression of op

IRI http://parliament.uk/ontologies/written-statement/expressionOf
Is Defined By pws:
Description

Relates a written statement expression to the written statement work realised by that expression. A written statement work has one written statement expression when a statement is made to one House. A written statement work has two written statement expressions when a statement is made to both Houses. A written statement expression realises one written statement work.
