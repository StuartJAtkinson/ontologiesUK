Contribution ontology

Classes · Object properties · Data properties · Namespaces · Depiction
A model describing contributions made by people or groups.

Classes

Agent

Class

An agent. The superset of all people and all groups of people.

Contribution

Class

A contribution by an individual or organisation to a document or committee inquiry etc.

ContributionRole

Class

The type of contribution made. Might be as an author or reviewer or inquiry witness etc.

Incumbency

Class

A position held by a person for a period of time. e.g. Philip Hammond occupying the position of Foreign Secretary between two dates.

Object properties

hasContributionRole

Object property

Relates a contribution to the type of contribution made.

Contribution — hasContributionRole — ContributionRole

inCapacity

Object property

Relates a contribution to an incumbency of the person making the contribution e.g. a person making a contribution in their capacity of holding a CEO position of an organisation.

Contribution — inCapacity — Incumbency

makesContribution

Object property

Relates a person or group to a contribution they've made.

Agent — makesContribution — Contribution

Data properties

Display order

Data property

Relates a contribution type to its display order.

ContributionType — Display order — integer

Is public

Data property

Relates a contribution to whether not that contribution is made public. The is public flag will by necessity not be public.

Contribution — Is public — boolean
