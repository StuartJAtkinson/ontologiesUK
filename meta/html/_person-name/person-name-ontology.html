Class CulturalOrdering

In using more meaningful semantic terms such as 'given name' rather than 'first name' we encounter cultural issues when the names are displayed in full - should the family name or given name be listed first? This class is a proposed way of allowing clients to understand that the naming in question is eg. from a culture where the given name comes before the family name

Class NamingQualification

A qualification or professional achievement which grants someone a pre-nominal or post-nominal decoration to their name, eg. attaining a doctorate allowing you to use the pre-nominal Dr or the post-nominal PhD

Class OfUsage

In a Peer's title, are they Baron Placename or Baron Of Placename

Class Peerage

This refers to the actual fact that a Person is awarded a Peerage. This should probably be modelled differently but is here a placeholder as of 8-6-17

Class PeerageAuthority

An authority that a peerage can be awarded in. e.g. an Scottish peerage or a UK peerage.

Class PeerageNaming

a type of PersonNaming that has been influenced by someone's peerage, whether that be through birth or through appointment to the House Of Lords

Superclasses · PersonNaming

Class PeerageRank

The peerage system is explicitly ranked, with a Baron being junior to an Earl. This level number will allow clients to display the more senior peergae where a peer has more than one peerage per Lords convention

Class PeerageType

Type of peerage, eg Life or Hereditary

Class PersonNaming

Could also be called 'identity' if that wasn't so ambiguous. ie People often have more than one naming over the course of their lives, from people changing their surnames at marriage to people being given appointments that have styles of address

Class PostNominal

A term or letters used after a person's name, eg. FRCOG

Class PreferredNaming

This subclassing is here to provide a pointer to clients as to which naming to use when otherwise not explicitly dictated by context

Class PreNominal
