Concept ontology - deprecated

Classes · Object properties · Data properties · Namespaces
Concept ontology for UK Parliament. Deprecated in favour of SKOS.

Classes

Agent

CLASS

A person or group. In this context a librarian or machine subject indexing annotatable things.

AnnotatableThing

CLASS

A thing which can be annotated with a concept or free text tag. e.g. a debate or petition or answer.

Annotation

CLASS

A relationship between an annotatable thing and a concept or free-text tag. e.g. an act of subject indexing.

Concept

CLASS

A concept. Similar to a SKOS concept / a library subject term.

ConceptScheme

CLASS

A concept scheme. Provides an arbitrary way to group concepts e.g. Commons Library to Lords Library.

Thing

CLASS

A thing. Anything at all.
