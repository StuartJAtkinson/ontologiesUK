Concept ontology for UK Parliament. Deprecated in favour of SKOS.

Classes

Agent class

A person or group. In this context a librarian or machine subject indexing annotatable things.

AnnotatableThing class

A thing which can be annotated with a concept or free text tag. e.g. a debate or petition or answer.

Annotation class

A relationship between an annotatable thing and a concept or free-text tag. e.g. an act of subject indexing.

Concept class

A concept. Similar to a SKOS concept / a library subject term.

ConceptScheme class

A concept scheme. Provides an arbitrary way to group concepts e.g. Commons Library to Lords Library.

Thing class

A thing. Anything at all.

Object properties

annotatedBy object property

Relates an annotation to a agent.

Annotation (domain) → annotatedBy (object property) → Agent (range)

hasAnnotation object property

Relates an annotatable thing to an annotation.

AnnotatableThing (domain) → hasAnnotation (object property) → Annotation (range)

hasBroaderConcept object property

Relates a concept to a broader concept.

Concept (domain) → hasBroaderConcept (object property) → Concept (range)

hasConcept object property

Relates an annotation to a concept.

Annotation (domain) → hasConcept (object property) → Concept (range)

hasFocus object property

Relates a concept to a real world thing that is its focus e.g. The concept / library subject term Winston Churchill to the person Winston Churchill.

Concept (domain) → hasFocus (object property) → Thing (range)

hasRelatedConcept object property

Relates a concept to another concept.

Concept (domain) → hasRelatedConcept (object property) → Concept (range)

inConceptScheme object property

Relates a concept to a concept scheme.

Concept (domain) → inConceptScheme (object property) → ConceptScheme (range)

Data properties

alternativeLabel data property

An alternative label.

Concept (domain) → alternativeLabel → langString (range)

annotatedAt data property

Date time an annotation was created.

Annotation (domain) → annotatedAt → dateTime (range)

historyNote data property

A history note. Textual description of editorial changes to a concept over time.

Concept (domain) → historyNote → langString (range)

indexerNote data property

A note for indexers. Provides additional guidance on use of the concept.

Concept (domain) → indexerNote → langString (range)

scopeNote data property

A scope note. Textual description of the intended usage of a concept.

Concept (domain) → scopeNote → langString (range)

uncontrolledTag data property

To allow for annotations that reference free text tags rather than controlled concepts.

Annotation (domain) → uncontrolledTag → langString (range)
