Procedure step ontology

Classes · Object properties · Data properties · Namespaces · Depiction
A model describing annotations added to procedure steps by the Indexing and Data Management Section of the House of Commons Library. Annotations are used to describe how a procedure step is actualised.

Classes

# Business step class

A business step is a step that can be actualised by a business item.

# Publication class

A publication.

Object properties

# Actualise with object property

Relates a business step to a business step where steps may be actualised together.

BusinessStep — Actualise with — BusinessStep

# Source object property

Relates a business step to the publication which is the source of information for that step.

BusinessStep — Source — Publication

Data properties

# Date note data property

A note that describes the date to use when actualising a business step.

BusinessStep — Date note — langString

# Link note data property

A note that describes where a business item should link to in a publication.

BusinessStep — Link note — langString

# Scope note data property

A note that describes the rules for actualising a business step.

BusinessStep — Scope note — langString
