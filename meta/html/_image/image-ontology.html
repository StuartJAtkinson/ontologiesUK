Image ontology

Ontology to describe storage and retrieval of parliamentary images

Classes#

Agent class**

An agent. The superset of all people and all groups of people.

GeoLocation class**

An area within a image, that will be used to identify multiple ImageSubjects

GeographicalThing (superclass) → GeoLocation (class)

Image class**

A image file, 2D unless otherwise stated, stored in the highest available quality

ImageCoordinate class**

A coordinate used to identify the location of the ImageZone within the Image

ImageCreator class**

The creator of the image - photographer, artist, designer of a diagram etc

Agent (superclass) → ImageCreator (class)

ImageOwner class**

The entity with the legal rights to the image and its use

Agent (superclass) → ImageOwner (class)

ImageResolution class**

An image's resolution

ImageSubject class**

A thing that appears in the image - what the image is at partially or wholly 'of'

ImageZone class**

An area within a image, that will be used to identify multiple ImageSubjects

Logo class**

A graphic mark, emblem, or symbol used to represent an organisation or person

Photograph class**

A Image taken with a camera

Image (superclass) → Photograph (class)

UsageLicence class**

The licence of the image's use, such as GNU or various Creative Commons licences
