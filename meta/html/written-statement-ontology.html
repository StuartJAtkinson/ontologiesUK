Written statement ontology

A model describing written statements made to Parliament as works and expressions of those works.

Classes · Object properties · Data properties · Namespaces · Depiction

Classes

Class Announced thing

A thing being announced by the making of a written statement to Parliament, for example: the laying of a paper.

Class Corrected thing

A thing being corrected by the making of a written statement to Parliament, for example: a preceding written statement or a written answer.

Class Document

A document.

Class House

The holding of a position by a person, including the time period for which this was the case. For example, Philip Hammond holding the position of Foreign Secretary between two dates.

Class Incumbency

The holding of a position by a person, including the time period for which this was the case. For example, Philip Hammond holding the position of Foreign Secretary between two dates.

Class Library deposit

A paper or set of papers placed by a Minister, or the Speaker, or the Lord Speaker, in the Library of either House. Usually deposits are made into both Libraries but sometimes only one.

Class Session

A period during which Parliament is able to conduct business on days of its choosing. A session is within a Parliament period. The first session of a Parliament begins on the same day as the Parliament period and ends on a day named in a subsequent prorogation proclamation. Subsequent sessions within the Parliament period begin and end on dates named in subsequent prorogation proclamations. The final session of a Parliament ends either with prorogation followed by dissolution or with dissolution. Session periods are the same for both Houses.

Class Written statement expression

The expression of a written statement as made to a House.

Class Written statement work

A written statement made to Parliament as realised in an expression made to a House.

Object property Expression of

Relates a written statement expression to the written statement work realised by that expression. A written statement work has one written statement expression when a statement is made to one House. A written statement work has two written statement expressions when a statement is made to both Houses. A written statement expression realises one written statement work.
