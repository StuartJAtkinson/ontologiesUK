Record ontology

A model informed by FRBR - describing works, collections of works, their expressions and manifestations.

Subclasses

record: Series
is a subclass of
record: Collection

Classes

Collection class

A curated collection of works.

Contribution class

A contribution by an agent, for example: by a person or by a corporate author.

Editorial form class

The editorial form of a work, for example: a report, an article, a transcript, or the minutes of a meeting.

Expression class

A work may be realised - as an expression - having more than one format, for example: a podcast episode made available as an audio file and as a text transcript.

File class

A file associated with a manifestation of a work, for example: a spreadsheet, a PDF or a Word document.

Focus class

A focus area for a manifestation or collection of works. Allows us to point at any modelled thing as a focus area, for example: a committee or a bill.

Format class

The format taken by the expression of a work, for example: the format of the expression of a podcast episode - as an audio file, or as a text transcript.

Manifestation class

A manifestation of an expression of a work, allowing for multiple versions of an expression over time, for example: an edition of a book, or a remastered audio file of a podcast episode.

Place class

A place named in a geography.

Revision class

A revision from a preceding to a following manifestation.

Revision type class

The type of a revision, for example: a major update or a minor update.

Series class

A specialised form of collection, allowing works to be placed in an order.

Collection (superclass) → Series (class)

Series position class

The position of a work within a series of works.

Work class

A distinct intellectual creation, similar to a work as defined in FRBR.

Object properties

Collection has focus object property

Relates a collection to the focus area of that collection.

Domain: Collection
Range: Focus

Collection part of collection object property

Relates a collection to a wider collection it forms part of, for example: a collection of works on health policy generally formed of collections of works concerned with particular areas of health policy.

Domain: Collection
Range: Collection

Expression of object property

Relates an expression to the work as expressed. An expression is of a single work. A work has one or more expressions.

Domain: Expression
Range: Work

From manifestation object property

Relates a revision to the manifestation which immediately precedes that revision.

Domain: Revision
Range: Manifestation

Has attachment object property

Relates a manifestation to a file..

Domain: Manifestation
Range: File

Has format object property

Relates an expression to the format taken by that expression. Different expressions of the same work may have different formats.

Domain: Expression
Range: Format

Has geographic scope object property

Denotes the geographic scope of a work, the scope being a geographic place.

Domain: Work
Range: Place

Has revision type object property

Relates a revision to the type of that revision.

Domain: Revision
Range: RevisionType

hasContribution object property

Relates a manifestation to a contribution. Contributors to a work may differ across different manifestations of that work.

Domain: Manifestation
Range: Contribution

In series object property

Relates the series position of a work to a series in which that work appears.

Domain: SeriesPosition
Range: Series

In series position object property

Relates a work to the position that work has within a series.

Domain: Work

Manifestation has focus object property

Relates a manifestation to the focus area of that manifestation.

Manifestation of object property

Relates a manifestation of an expression to that expression. An expression may have more than one manifestation. A manifestation has a single expression.

Domain: Manifestation
