Utility ontology

Classes · Object properties · Data properties · Subclasses · Namespaces · Depiction
Provides a set of common properties for reuse by other models.

Classes

Identifier

CLASS

An identifier from an external system. Might be another Parliament system or an external system such as Wikidata.

Identifier source

CLASS

An identifier source. The system or website from which the identifier originates.

Label

CLASS

Allows a thing to have different labels at different time periods. e.g. a Government Department, Select Committee, Bill or inquiry might change its name during its lifetime. Reuses preferredLabel, abbreviationLabel, sortLabel, startDate and endDate from Thing.

Temporal thing

CLASS

An interval between two points in time, having a name.

Superclasses · Thing

Thing

CLASS

A thing. Anything. At all. Provides a set of properties for reuse by any other model.

Web link

CLASS

A web link with a source, type and URL.

Web link source

CLASS

The publisher of the URL identified by the web link e.g. The Daily Mail, personal website, Facebook, Wikipedia.

Web link type

CLASS

A link type e.g. personal homepage, obituary.
