utility/utility-ontology.ttl

Utility ontology

Provides a set of common properties for reuse by other models.

Subclasses

Classes

Identifier class

An identifier from an external system. Might be another Parliament system or an external system such as Wikidata.

IdentifierSource class

An identifier source. The system or website from which the identifier originates.

Label class

Allows a thing to have different labels at different time periods. e.g. a Government Department, Select Committee, Bill or inquiry might change its name during its lifetime. Reuses preferredLabel, abbreviationLabel, sortLabel, startDate and endDate from Thing.

Thing class

A thing. Anything. At all. Provides a set of properties for reuse by any other model.

WebLink class

A web link with a source, type and URL.

WebLinkSource class

The publisher of the URL identified by the web link e.g. The Daily Mail, personal website, Facebook, Wikipedia.

WebLinkType class

A link type e.g. personal homepage, obituary.

Object properties

hasIdentifier object property

Relates a thing to an identifier.

Domain: Thing
Range: Identifier

hasIdentifierSource object property

Relates an identifier to its source.

Domain: Identifier
Range: IdentifierSource

hasLabel object property

Relates a thing to a label during a time period.

Domain: Thing
Range: Label

hasWebLink object property

Relates a thing to a web link. e.g. Person to a twitter account.

Domain: Thing
Range: WebLink

hasWebLinkScource object property

Relates a web link to a link source.

Domain: WebLink
Range: WebLinkSource

hasWebLinkType object property

Relates a link to a weblink type.

Domain: WebLink
Range: WebLinkType

precededBy object property

Relates a thing to its predecessor. Usually the thing and its preceding thing will be of the same class.

Domain: Thing
Range: Thing

Data properties

abbreviationLabel data property

Relates a thing to an abbreviation label.

Thing (domain) → abbreviationLabel → langString (range)

createdAt data property

Relates a thing to a created at date / time.

Thing (domain) → createdAt → dateTime (range)

description data property

Relates a thing to its textual description.

Thing (domain) → description → langString (range)

endOn data property

Relates a thing to an end date.

Thing (domain) → endOn → date (range)

identifierValue data property

Relates an identifier to its value.

Identifier (domain) → identifierValue → Literal (range)

preferredLabel data property

Relates a thing to its preferred label.

Thing (domain) → preferredLabel → langString (range)

Recorded end at data property

Relates a thing to its recorded end time.

Thing (domain) → Recorded end at → dateTime (range)

Recorded start at data property

Relates a thing to its recorded start time.

Thing (domain) → Recorded start at → dateTime (range)

sameAs data property

Relates a thing to the same thing described elsewhere on the web. Initially intended for wikidata NIR URIs.

Thing (domain) → sameAs → Literal (range)

Scheduled end at data property

Relates a thing to its scheduled end time.

Thing (domain) → Scheduled end at → dateTime (range)

Scheduled start at data property

Relates a thing to its scheduled start time.

Thing (domain) → Scheduled start at → dateTime (range)

sortLabel data property

Relates a thing to a sort label e.g. Churchill, Winston.

Thing (domain) → sortLabel → langString (range)

sortOrder data property

Relates a thing to a sort order.

Thing (domain) → sortOrder → integer (range)

startOn data property

Relates a thing to a start date.

Thing (domain) → startOn → date (range)

updatedAt data property

Relates a thing to an updated at date / time.

Thing (domain) → updatedAt → dateTime (range)

url data property

Relates a web link to a url.

WebLink (domain) → url → Literal (range)
