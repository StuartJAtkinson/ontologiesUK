Stats series ontology - deprecated

Statistical series ontology for UK Parliament. Primarily aimed at modelling statistical information around places. Deprecated in favour of RDF Data Cube.

Classes

AverageType class

Used to express the method of averaging used for data set releases covering a time period rather than a point in time (date).

Businesses class

A population of businesses in a place. This might be all registered businesses or business in a certain industrial sector.

Population (superclass) → Businesses (class)

DataPoint class

A specific data point in a statistics release version. A measure for a given population at a point in time or averaged over an interval.

DataSet class

A data set e.g. age profiles by constituency over time.

DataSetRelease class

A release of a data set. Different versions may be published over time to correct mistakes.

DataSetReleaseVersion class

A version of a release of a data set. Different versions may be released over time to correct miscalculations etc.

People class

A population of people in a place. This might be all people resident or over 70s resident or people in receipt of Universal Credit etc.

Population (superclass) → People (class)

Period class

The period over which a measurement is counted e.g. wages per week.

Place class

A place. Any geographical extent. In the case of parliamentary demographic statistics we'll usually be dealing with a constituency area but other places may be needed in the future e.g. local authority areas.

Population class

A set of things bounded by a geographical extent. Might be a population of residents, a population of those eligible to vote, a population of small businesses, a population of static caravans, in a constituency etc.

Residences class

A population of residences in a place. This might be all residences or local authority owned residences etc.

Population (superclass) → Residences (class)

Source class

The source organisation for a given subset of a data set release. Much of the data published by the House of Commons Library originates with the Office of National Statistics but often originates with other organisations for the devolved nations e.g. Northern Ireland Statistics and Research Agency for Northern Ireland. Other data sets originate in government departments.

TemporalCoverage class

The period or point in time (date) covered by a statistics release e.g. quarter 1 of 2018 or April 2016. Allows period coverage to vary between releases.

TerritorialApplicability class

The territorial applicability of a data set release. Allows us to say that a given release of a data set contains data from a given source (organisation) for a given place e.g. a constituency. Sources may change between releases.

Unit class

The unit of measurement for all data points in a stats series e.g. GBP.

Object properties

averagedOverPeriodAs object property

Relates a temporal coverage to any means of averaging employed on the data.

TemporalCoverage (domain) → averagedOverPeriodAs (object property) → AverageType (range)

boundBy object property

Relates a population (of people, residences, businesses etc) to a geographic area.

Population (domain) → boundBy (object property) → Place (range)

countPer object property

Relates a data set to the period over which the data points in the data set are counted e.g. average wage per week.

DataSet (domain) → countPer (object property) → Period (range)

fromSource object property

Relates territorial applicability to a data source organisation.

TerritorialApplicability (domain) → fromSource (object property) → Source (range)

hasTerritorialApplicability object property

Relates a data set release to it's territorial applicability.

DataSetRelease (domain) → hasTerritorialApplicability (object property) → TerritorialApplicability (range)

hasUnit object property

Relates a data set to the unit in which the data is counted e.g. GBP.

DataSet (domain) → hasUnit (object property) → Unit (range)

inDataSetReleaseVersion object property

Relates a data point to the data set release version it forms part of.

DataPoint (domain) → inDataSetReleaseVersion (object property) → DataSetReleaseVersion (range)

inPlace object property

Relates territorial applicability to a geographic area.

TerritorialApplicability (domain) → inPlace (object property) → Place (range)

measureOf object property

Relates a data point to a given population.

DataPoint (domain) → measureOf (object property) → Population (range)

releaseOf object property

Relates a data set release to a data set it forms part of.

DataSetRelease (domain) → releaseOf (object property) → DataSet (range)

releaseVersionOf object property

Relates a data set release version to a data set release.

DataSetReleaseVersion (domain) → releaseVersionOf (object property) → DataSetRelease (range)

temporalCoverage object property

Relates a data set release to the time period or point in time the data was gathered.

DataSetRelease (domain) → temporalCoverage (object property) → TemporalCoverage (range)

Data properties

caveatText data property

Relates a data set to any caveat text accompanying it.

DataSet (domain) → caveatText → langString (range)

endOn data property

Relates a temporal coverage of a period to the end date of that period.

TemporalCoverage (domain) → endOn → date (range)

onDate data property

Relates a temporal coverage of a day to that date.

TemporalCoverage (domain) → onDate → date (range)

releasedOn data property

Relates a data set release version to the date on which it was released.

DataPoint (domain) → releasedOn → date (range)

releaseNotes data property

Relates a data set release to explanatory notes.

DataSetRelease (domain) → releaseNotes → langString (range)

releaseVersionNotes data property

Relates a data set release version to explanatory notes.

DataSetReleaseVersion (domain) → releaseVersionNotes → langString (range)

startOn data property

Relates a temporal coverage of a period to the start date of that period.

TemporalCoverage (domain) → startOn → date (range)

updateFrequency data property

Relates a data set to any textual account of its update frequency.

DataSet (domain) → updateFrequency → langString (range)

value data property

Relates a data point to it's value as an integer.

DataPoint (domain) → value → integer (range)
