Stats series ontology - deprecated

Statistical series ontology for UK Parliament. Primarily aimed at modelling statistical information around places. Deprecated in favour of RDF Data Cube.

A set of things bounded by a geographical extent. Might be a population of residents, a population of those eligible to vote, a population of small businesses, a population of static caravans, in a constituency etc.

Class LABEL PLACEHOLDER

A place. Any geographical extent. In the case of parliamentary demographic statistics we'll usually be dealing with a constituency area but other places may be needed in the future e.g. local authority areas.

Class LABEL PLACEHOLDER

A specific data point in a statistics release version. A measure for a given population at a point in time or averaged over an interval.

Class LABEL PLACEHOLDER

A version of a release of a data set. Different versions may be released over time to correct miscalculations etc.

Class LABEL PLACEHOLDER

A release of a data set. Different versions may be published over time to correct mistakes.

Class LABEL PLACEHOLDER

A data set e.g. age profiles by constituency over time.

Class LABEL PLACEHOLDER

The territorial applicability of a data set release. Allows us to say that a given release of a data set contains data from a given source (organisation) for a given place e.g. a constituency. Sources may change between releases.

Class LABEL PLACEHOLDER

The source organisation for a given subset of a data set release. Much of the data published by the House of Commons Library originates with the Office of National Statistics but often originates with other organisations for the devolved nations e.g. Northern Ireland Statistics and Research Agency for Northern Ireland. Other data sets originate in government departments.

Class LABEL PLACEHOLDER

The period or point in time (date) covered by a statistics release e.g. quarter 1 of 2018 or April 2016. Allows period coverage to vary between releases.

Class LABEL PLACEHOLDER

Used to express the method of averaging used for data set releases covering a time period rather than a point in time (date).

Class LABEL PLACEHOLDER

The unit of measurement for all data points in a stats series e.g. GBP.

Class LABEL PLACEHOLDER

The period over which a measurement is counted e.g. wages per week.
