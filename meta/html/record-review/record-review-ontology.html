Record reviews ontology

Classes · Object properties · Data properties · Subclasses · Namespaces · Depiction
A model describing editorial reviews of records and actions taken following review.

Classes

Agent

CLASS

Agents include people and groups, including those outside the Parliamentary Service, participating in parliamentary procedure and contributing to the wider work of Parliament.

Archiving

CLASS

The action of archiving a work, indicating that no further revisions are intended to take place. Manifestations of the work which have not been withdrawn remain available, marked with an indication that no further updates are anticipated.

Superclasses · EditorialAction

Editorial action

CLASS

An editorial action is an action taken on a manifestation or a work, for example: archiving a work, reviewing a manifestation, revising a manifestation or withdrawing a manifestation.

Editorial review

CLASS

A review of a manifestation. A decision to review may be triggered by an external event, for example: a change in Government policy, the availability of new data, external work in the same area or the output of parliamentary business; a change in user needs, for example: a new type of enquiry; or a change to library policy or practice, for example: in presentation, citation style, method of acknowledgement, change of portfolio or change of specialist.

Superclasses · EditorialAction

Feedback

CLASS

Feedback on a record, provided in a submission. An instance of the feedback class may contain the text of the feedback, or a pointer to the substance of the feedback in a separate system.

Manifestation

CLASS

A manifestation of an expression of a work, allowing for multiple versions of an expression over time, for example: an edition of a book, or a remastered audio file of a podcast episode.

Review cycle

CLASS

A review cycle describes the period between reviews of a work.

Revision

CLASS

The action of revising an existing manifestation, resulting in a new manifestation. The existing manifestation remains unchanged.

Superclasses · EditorialAction

Revision type

CLASS

The type of a revision, for example: a major or minor edit.

Withdrawal

CLASS

The action of withdrawing a manifestation. A manifestation may be withdrawn for a period and reinstated, or permanently withdrawn.

Superclasses · EditorialAction

Work

CLASS

A distinct intellectual creation, similar to a work as defined in FRBR.
