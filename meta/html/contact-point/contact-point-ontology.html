Classes

ContactableThing class

A thing that can be contacted. Might be a person or a person in the context of a role or a group.

ContactPoint class

A postal address or other contact details.

PostalAddress class

A postal address.

Object properties

hasContactPoint object property

Relates a contactable thing (a person, group or position etc) to a contact point.

ContactableThing (domain) → hasContactPoint (object property) → ContactPoint (range)

hasPostalAddress object property

Relates a contact point to its postal address.

ContactPoint (domain) → hasPostalAddress (object property) → PostalAddress (range)

availabilityNote data property

Temporal availability note for a contact point. Used for constituency office opening days and hours.

ContactPoint (domain) → availabilityNote → langString (range)

contactForm data property

URL for a contact form for contacting a contact point. Instead of an email address e.g. some government departments publish a contact form rather than exposing email addresses.

ContactPoint (domain) → contactForm → Literal (range)
