Classes

CalendarEvent class

A planned or arranged event in a place.

Call class

A call by a committee or committees conducting an inquiry for written evidence, oral evidence, papers or records.

Event (superclass) → Call (class)

CallForEvidence class

An open call by a committee or committees conducting an inquiry for members of the public or organisations to provide written evidence. Sometimes referred to as a call for written submissions.

Call (superclass) → CallForEvidence (class)

CallForOralEvidence class

A call by a committee or committees conducting an inquiry for a named witness to provide oral evidence.

Call (superclass) → CallForOralEvidence (class)

CallForPapers class

A call by a committee or committees conducting an inquiry for a named individual or organisation to provide papers. Papers are understood to be documents that the individual or organisation are unlikely to possess but would be able to compile.

Call (superclass) → CallForPapers (class)

CallForRecords class

A call by a committee or committees conducting an inquiry for a named individual or organisation to provide records. Records are understood to be documents that the individual or organisation are known to possess.

Call (superclass) → CallForRecords (class)

Compelled class

A call by a committee or committees conducting an inquiry compelling a named witness to provide oral evidence.

CallForOralEvidence (superclass) → Compelled (class)

Contribution class

A contribution by an individual or organisation to a committee inquiry. This might be in the form of written evidence, oral evidence, papers or records.

Document class

A document.

Event class

An event. A thing that happens over a time interval.

Focus class

A focus area for a contribution. Borrowed from the formal body ontology to identify the subject area or areas covered by a contribution.

Incumbency class

A position held by a person for a period of time e.g. Philip Hammond occupying the position of Foreign Secretary between two dates.

Person class

A person.

Object properties

callForOralEvidence object property

Relates a call for oral evidence to a contribution made in response to that call.

CallForOralEvidence (domain) → callForOralEvidence (object property) → Contribution (range)

callForPapers object property

Relates a call for papers to a contribution made in response to that call.

CallForPapers (domain) → callForPapers (object property) → Contribution (range)

callForRecords object property

Relates a call for records to a contribution made in response to that call.

CallForRecords (domain) → callForRecords (object property) → Contribution (range)

compelledToAttend object property

Relates a compelled call for oral evidence to a calendar event where the witness is expected to be heard.

Compelled (domain) → compelledToAttend (object property) → CalendarEvent (range)

contributes object property

Relates a contribution to a document contributed.

Contribution (domain) → contributes (object property) → Document (range)

focusOfContribution object property

Relates a contribution to its focus (subject) area(s).

Contribution (domain) → focusOfContribution (object property) → Focus (range)

inCapacity object property

Relates a contribution to the incumbency of the person making the contribution e.g. a person making a contribution in their capacity of holding a CEO position of an organisation.

Contribution (domain) → inCapacity (object property) → Incumbency (range)

makesContribution object property

Relates a person making a contribution to their contribution.

Person (domain) → makesContribution (object property) → Contribution (range)

respondsToCall object property

Relates any type of contribution to the call it is submitted in response to.

Contribution (domain) → respondsToCall (object property) → Call (range)

Data properties

callDescription data property

Relates any type of call to its descriptive text.

Call (domain) → callDescription → langString (range)

calledAt data property

The date time at which a call is made.

Call (domain) → calledAt → dateTime (range)

callEnds data property

The date time at which a call ends.

Call (domain) → callEnds → dateTime (range)
