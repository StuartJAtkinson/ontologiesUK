Submission ontology

Ontology to describe calls from Parliament for submission of evidence and resulting submissions.

Classes

CalendarEvent class

A planned or arranged event in a place.

Call class

A call by a committee or committees conducting an inquiry for written evidence, oral evidence, papers or records.

Event (superclass) → Call (class)

CallForEvidence class

An open call by a committee or committees conducting an inquiry for members of the public or organisations to provide written evidence. Sometimes referred to as a call for written submissions.

Call (superclass) → CallForEvidence (class)

CallForOralEvidence class

A call by a committee or committees conducting an inquiry for a named witness to provide oral evidence.

Call (superclass) → CallForOralEvidence (class)

CallForPapers class

A call by a committee or committees conducting an inquiry for a named individual or organisation to provide papers. Papers are understood to be documents that the individual or organisation are unlikely to possess but would be able to compile.

Call (superclass) → CallForPapers (class)

CallForRecords class

A call by a committee or committees conducting an inquiry for a named individual or organisation to provide records. Records are understood to be documents that the individual or organisation are known to possess.

Call (superclass) → CallForRecords (class)

Compelled class

A call by a committee or committees conducting an inquiry compelling a named witness to provide oral evidence.

CallForOralEvidence (superclass) → Compelled (class)

Contribution class

A contribution by an individual or organisation to a committee inquiry. This might be in the form of written evidence, oral evidence, papers or records.

Document class

A document.

Event class

An event. A thing that happens over a time interval.

Focus class

A focus area for a contribution. Borrowed from the formal body ontology to identify the subject area or areas covered by a contribution.

Incumbency class

A position held by a person for a period of time e.g. Philip Hammond occupying the position of Foreign Secretary between two dates.

Person class

A person.
