AllocatedAnsweringBodyc back to ToC or Class ToC

IRI: http://parliament.uk/ontologies/written-question-and-answer/AllocatedAnsweringBody

The answering body allocated to a question, either at tabling or in the event of the question being transferred. Note that the allocated answering body can change over time when one answering body transfers responsibility to another answering body. So allocated answering bodies are time bound. This would use startAt and endAt properties from the Core ontology.
is in domain of
allocatedToop
is in range of
allocationop, transferredFromop, transferredToop

AnswerTypec back to ToC or Class ToC

IRI: http://parliament.uk/ontologies/written-question-and-answer/AnswerType

The type of a given answer e.g. holding answer, current answer or corrected answer. If a question is asked toward the end of a session, and Parliament is prorogued before it can be answered, a prorogation answer may be provided. The answer will state that there was not time to answer the question before prorogation.
is in range of
hasAnswerTypeop, previousAnswerop

WrittenQuestionc back to ToC or Class ToC

IRI: http://parliament.uk/ontologies/written-question-and-answer/WrittenQuestion

A type of Parliamentary question to which a written answer is expected. Written questions allow members to ask for information on the work, policy and activities of answering bodies. Usually this is to a minister in a government department but sometimes questions can be asked of non-departmental bodies. e.g. Church Commissioners.

answerToop back to ToC or Object Property ToC

IRI: http://parliament.uk/ontologies/written-question-and-answer/answerTo

Allows a written answer to be associated with one or more written questions. This replicates more precisely the current functionality around grouped questions. In other words, a group of questions in receipt of the same answer. There is some debate as to whether the same text given in answer to questions from multiple members can be considered one, or the same, answer. This also allows a single question to have many answers. Sometimes multiple answers from a single answering body (e.g. holding answer, current answer, correction). Sometimes different answers from different answering bodies to the same question e.g. answers to a round robin question. Note that this is a major change to current data models, where a single question can have one and only one answer, and a single answer can be to one and only one question.
has domain
WrittenAnswerc
has range
WrittenQuestionc

IRI: http://parliament.uk/ontologies/written-question-and-answer/namedDateForAnswer

Relates a written question to a named day. The member tabling the question has requested that it be answered by the named day. Note that named day questions can only be tabled in the House of Commons. Most written questions are 'ordinary'. By convention a member can expect such questions to be answered within 7 days of tabling. Named day questions are used to elicit a faster response.
